Definiciones

Kubernetes es un orquestador de contenedores. Permite gestionar la carga del tráfico, configurar enrutamientos, controlar la escala, etc. de varios servidores (contenedores) a la vez.

Terraform es una herramienta que permite definir infraestructura como código, es decir, de forma declarativa.

OpenTofu es un fork de Terraform, que comenzó cuando la empresa detrás del producto, HashiCorp (ahora adquirida por IBM), decidió cambiar la licencia a closed source.

Grafana es una herramienta de observabilidad que permite centralizar métricas y registros (logs) en una sola plataforma para desarrollar todo tipo de visualizaciones y alarmas.

Prometheus es un servidor que recolecta y acumula métricas de múltiples lugares, que se puede integrar con Grafana para potenciar el monitoreo de servidores.
